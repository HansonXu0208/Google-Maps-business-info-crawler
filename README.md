# Google-Maps-business-info-crawler
## How to Use
This application uses the Google Maps Places API to search and extract business information.  
Follow the steps below to get started:

### 1. Get a Google Maps API Key
- Go to: https://console.cloud.google.com/
- Sign in with your Google account
- Create a new project
- Navigate to **APIs & Services → Library**
- Enable the following APIs:
  - Maps JavaScript API
  - Places API

### 2. Create an API Key
- Go to **APIs & Services → Credentials**
- Click **Create Credentials → API Key**
- Copy your API key

### 3. Run the Application
- Open the `index.html` file in your browser  
  OR  
- Use the GitHub Pages link if deployed

### 4. Enter Your API Key
- Paste your Google API key into the input field in the application

### 5. Search for Businesses
- Enter a location (e.g., Melbourne)
- Set a search radius
- Input keywords (e.g., "restaurant", "construction", "stone supplier")
- Click **Search**

### 6. Export Data
- After results are displayed, click **Export CSV**
- The data will be downloaded for further analysis or lead generation

## Notes
- This tool depends on Google Maps Platform APIs, which may incur usage costs
- Make sure billing is enabled in your Google Cloud account
- Do NOT expose your API key publicly in the code
